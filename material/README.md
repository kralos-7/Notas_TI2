# ☕ Sistema de Cafetería Universitaria

Aplicación web desarrollada para gestionar pedidos en una cafetería universitaria.  
Permite a los usuarios ver el menú, realizar pedidos y a los administradores gestionar productos.

## 🧠 Descripción

Este proyecto fue desarrollado como práctica de desarrollo frontend y backend utilizando tecnologías modernas.

El sistema incluye:
- Visualización de productos
- Carrito de compras
- Gestión de pedidos
- Panel de administración

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Node.js
- Express
- EJS

## 📂 Estructura del proyecto

```

proyecto/
│
├── public/ # Archivos estáticos (CSS, JS, imágenes)
````

## ⚙️ Instalación

Sigue estos pasos para ejecutar el proyecto en tu computadora:

```bash
# Clonar repositorio
git clone https://github.com/usuario/proyecto.git

# Entrar al proyecto
cd proyecto

# Instalar dependencias
npm install

# Ejecutar servidor
npm start
````

---

## 💻 Uso

1. Abre el navegador en:
   [http://localhost:3000](http://localhost:3000)

2. Explora el menú y realiza pedidos

3. Accede al panel admin (si aplica)
