# Tecnologías de Información II
**Licenciatura en Informática | Universidad del Mar (UMAR)**

Este repositorio contiene el material teórico, casos prácticos y arquitecturas de referencia de la asignatura de Tecnologías de Información II.

## 📂 Estructura del Repositorio
* **/docs**: Notas de clase, presentaciones en LaTeX (Beamer/Metropolis) y lecturas académicas.
  * `week_1/`: Introducción, Modelos de E-commerce, Arquitectura de Plataformas y Caso Práctico.
* **/project**: (Próximamente) Enunciados y recursos para el proyecto integrador.

## 🛠️ Tecnologías y Conceptos Clave
* **Arquitecturas:** Headless Commerce, Microservicios, Event-Driven Design.
* **Resiliencia:** Patrones Circuit Breaker, Message Queuing (Kafka/RabbitMQ) y Observabilidad.
* **Ecosistemas:** Economía de APIs, Estrategias Multi-tenant y Efectos de Red.
* **Seguridad:** Cumplimiento PCI-DSS, Tokenización y Prevención de Fraude en Tiempo Real.
